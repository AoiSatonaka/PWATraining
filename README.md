# PWATraining

## 目的

- PWAを使用して、アプリケーションを作成するための下積み。

## 環境構築

### Sass

```shell
> sudo gem install sass
```

### pug

```shell
> npm i -D pug pug-cli
```

### npm-run-all

```shell
> npm i -D npm-run-all
```

#### usages

```shell
# ----- sass -----
> sass [source]:[out]

# ----- pug -----
> npx pug [source] --out [outdir] [options...]


# ----- npm-run-all -----
> {npm-run-all,run-{p,s}} [target] [options...]
```
